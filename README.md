# smplkit

Official Python SDK for the [smplkit](https://www.smplkit.com) platform.

## Installation

```bash
pip install smplkit-sdk
```

## Quick Start

```python
from smplkit import SmplkitClient

client = SmplkitClient(api_key="sk_api_...")
```
